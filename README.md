# student-service

## for getting data of students
http://127.0.0.1:7000/student/1

## for create data of students
http://127.0.0.1:7000/student
and request body data format
{
    "studentId":"1",
    "name":"tarun gupta"
}

## for update data
http://127.0.0.1:7000/student/1
and request body
{
    "studentId":"1",
    "name":"tarun gupta"
}

## for deleting data
http://127.0.0.1:7000/student/1
